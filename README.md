# 🎯 Guess The Number Game (Java)

A simple console-based **Guess The Number Game** built using Java.

## 📌 Features

- Random number generation
- User input using Scanner
- Unlimited guessing attempts
- Hint if the guess is too high or too low
- Displays the number of attempts taken
- Simple and beginner-friendly code

## 🛠️ Technologies Used

- Java
- IntelliJ IDEA
- Object-Oriented Programming (OOP)

## ▶️ How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Vikashkush2026/number-guessing-game-java.git
   ```

2. Open the project in IntelliJ IDEA.

3. Run the `Game1.java` file.

## 📷 Sample Output

```
Welcome to Guess The Number Game!

Enter your guess: 50
Too High!

Enter your guess: 25
Too Low!

Enter your guess: 37
🎉 Congratulations! You guessed the correct number.
Attempts: 3
```
